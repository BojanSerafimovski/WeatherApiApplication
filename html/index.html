Welcome to my Weather API Project

Please have in mind that if you're having any troubles with loading the API data, you may need to activate your own API key!

On this project the users can search for a city/state/continent and check the weather conditions at the moment.
The custom properties for my project are set for the city I live in which is Skopje.
Thank you for being here! Enjoy the forecast and the randomized backgrounds depending on the weather description.
